I designed and implemented a robust relational database for a modern e-commerce platform named "AfriShop". The main objective was to create a clean database structure capable of handling real-time stock management, customer orders, and secure payment tracking simulation using international gateways like stripe and payPal AOIs.

Key features of my work on this project:
1. Conceptual Data Modeling: Developed a clean Entity-Relationship Diagram (ERD) ensuring correct table relations.
2. Data Integrity: Enforced strict SQL constraints (PRIMARY KEY, UNIQUE transaction references, and NOT NULL fields).
3. Advanced Business Queries: Wrote complex SQL scripts to calculate total revenue, generate low-stock alerts, and track customer spending habits.

The entire backend architecture was successfully built and tested using MySQL and phpMyAdmin.
